## Objetivo do app

O Gabarita e um app de estudos para resolver questoes, fazer simulados, acompanhar evolucao e revisar pontos fracos.

O foco do projeto e demonstrar:

- organizacao em camadas;
- persistencia local com SQLite;
- gerenciamento de estado com Provider;
- uso de sensores e recursos nativos;
- interface moderna em Flutter;
- funcionamento offline com banco local de questoes.

## Estado atual

O app possui:

- banco local de questoes do ENEM em JSON;
- importacao local para SQLite;
- telas principais implementadas;
- providers conectando UI e regras de negocio;
- comentarios explicativos no codigo;
- documentacao em Markdown para GitHub.
